Programas open source para pintar digital y editar imágenes

Programas open source para pintar digital son la alternativa a los programas comerciales.

Lo programas comerciales para arte en tu computadora son costosos. Gracias a los programas open source (código abierto) el costo no es un problema. Con esta selección podrás pintar hermosas obras digitales.

https://www.gimp.org/

GIMP es un editor de imágenes multiplataforma disponible para GNU / Linux, OS X, Windows y más sistemas operativos. Es software libre, puedes cambiar su código fuente y distribuir tus cambios. Si usted es un diseñador gráfico, fotógrafo, ilustrador o científico, GIMP le proporciona herramientas sofisticadas para hacer su trabajo. Puede mejorar aún más su productividad con GIMP gracias a muchas opciones de personalización y complementos de terceros.

www.Krita.org  Disponible para Windows,linux y OSX

Krita es un programa profesional de pintura digital, gratuito y hecho con código libre, ha sido creado por artistas que desean hacer éstas herramientas accesibles para todos.

  • Arte conceptual
  • Pinturas de textura y mate
  • Ilustraciones e historietas

https://medibangpaint.com/es/ Windows/ Mac/ iPad/ iPhone/ Android

Medibang Paint es un programa para la creación de cómics e ilustración digital, liviano y GRATUITO. Tiene pinceles, fuentes, fondos prediseñados y otros recursos.

http://al.chemy.org/  Disponible para Windows,linux y OSX

Alchemy es un proyecto de dibujo abierto que pretende explorar cómo podemos dibujar, dibujar y crear en computadoras de nuevas maneras. Alchemy no es un software para crear obras terminadas, sino un entorno de bosquejo que se centra en la etapa inicial absoluta del proceso de creación.

http://firealpaca.com/es  Disponible para Windows y OSX

FireAlpaca es un software de pintura de uso simple y sencillo.

Bono: Este es un programa comercial pero cuenta con una versión de uso gratuito

https://www.madewithmischief.com/ Disponible para Windows y OSX

¡Herramientas esenciales con un lienzo infinito, perfecto para sacar ideas y los doodles sin fin!

¡A crear con libertad!
