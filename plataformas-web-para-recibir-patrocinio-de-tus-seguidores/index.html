Plataformas web para recibir patrocinio de tus seguidores

Estas herramientas facilitan a tus seguidores la tarea de enviar su donativo a través de sus plataformas web.

Desarrollar obra artística o contenido multimedios requiere tiempo, trabajo y dinero. Nunca está de mas recibir algún apoyo financiero para poder dedicar mas tiempo a nuestro trabajo. Si cuentas con una buena base de fans que aman tu trabajo puedes recibir su apoyo económico para continuar tu obra.
Hoy te presentamos algunas páginas webs que proveen el servicio:

Patreon

Una plataforma bastante conocida y muy completa. Se enfoca en la membresía donde tus suscriptores pagan por ver contenido exclusivo. Tiene una comisión del 5%

Las siguientes plataformas usan el concepto de invitar un café digital, que no es otra cosa que un pequeña donación monetaria, que puede ser de una vez o recurrente.

Buymeacoffee

Puedes crear un link para compartir o para colocar en una pagina web donde tus seguidores podrán entrar para donar. Es la plataforma mas sencilla de las reseñadas..

Beabacker

Sigue el concepto de invitar un café digital y junto con la donación puedes recibir un mensaje de apoyo. Cuenta con un panel de control donde puedes ver todas las donaciones recibidas, es totalmente gratis y no toman comisión. Necesitas una cuenta de Stripe para recibir las donaciones y poder publicar tu perfil. El donante puede establecer pagos recurrentes y siempre tendrá que dar una propina de $1 a beabacker.

ko-fi

También bajo el concepto de café digital. Parecida a una red social contiene opción de galería y contenido exclusivo para suscriptores. Ko-fi es mas parecido a una red social y es totalmente gratis y sin comisión. Este puede ser el mas conveniente para artistas visuales por su función de galería y puedes hacer publicaciones Blog para tus seguidores. Te registras como creator o supporter y conectas una cuenta de paypal para recibir las donaciones. También cuenta con una membresía Gold con prestaciones adicionales como pagos recurrentes y contenido exclusivo.

Espero que puedas sacar provecho de alguna de estas herramientas y te ayuden a generar algunos ingresos. Cuéntanos como te fue.
