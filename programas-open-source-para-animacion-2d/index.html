Programas open source para animación 2d

Programas open source para animación 2d sin limitaciones.

Este es un listado de los programas open source (código abierto) mas reconocidos en el campo de la animación 2d. Con estos programas podrás realizar esas animaciones 2d que tanto te gustan.

http://www.pencil2d.org/

Pencil2D es un software de animación / dibujo para Mac OS X, Windows y Linux. Le permite crear animaciones tradicionales dibujadas a mano (dibujos animados) utilizando mapas de bits y gráficos vectoriales. El lápiz es libre y de código abierto.

http://www.synfig.org/

Disponible para Windows y OSX. Software de animación 2D en código abierto

https://opentoonz.github.io/e/index.html

Disponible para Windows y OSX. Software para producir una animación 2D

http://www.maefloresta.com/tupi/es/acerca

Windows,linux y OSX

Una herramienta de diseño y creación para artistas digitales interesados en animación 2D, ofreciendo una experiencia de interfaz enfocada a niños de 8 a 100 años. Un proyecto de Software Libre en constante re-diseño y evolución, liberado bajos los términos de la licencia GPL (versión 2+)

Bono: Programa para crear historias interactivas

https://www.renpy.org/

Windows,linux y OSX

Ren'Py es un motor de novelas visuales que le ayuda a usar palabras, imágenes y sonidos para contar historias interactivas que se ejecutan en computadoras y dispositivos móviles. Estos pueden ser tanto novelas visuales y juegos de simulación de vida.

¡A crear con libertad!
